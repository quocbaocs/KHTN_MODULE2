# XML DOM
	- Dùng để chứa và truyền dữ liệu

## Cú pháp XML
	
	1. Có một nút gốc
	2. Phải có tag mở/đóng
	3. tag mở/đóng phải trùng tên
	4. tag con nằm trong tag cha
	5. Giá trị thuộc tính phải đặt trong "chuỗi"
	
## Cú pháp JSON

	{"ten":"gia tri",...}
	{"ten":[{"ten":"gia tri",....}]}
